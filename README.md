# 💰 Expense Tracker Web App

A modern and responsive **Expense Tracker Web Application** that helps users manage their daily finances efficiently.  
Track income, monitor expenses, and gain insights into your spending habits — all in one place.

🔗 **Live Demo:** https://expense-tracker-neon-nu.vercel.app/

---

## 🚀 Features

- 💵 Track income and expenses in real-time
- 📊 Dashboard with financial overview
- 📋 View all transactions in one place
- ➕ Add new income and expense entries
- 🧾 Categorize transactions for better tracking
- 📱 Fully responsive design (mobile + desktop)
- ⚡ Fast and smooth user experience

---

## 🛠️ Tech Stack

- **Frontend:** React Js
- **Styling:** Tailwind CSS / CSS
- **Deployment:** Vercel
- **State Management:** React Hooks
